BTA

BTA is an Application tool designed to provide teachers and learners with WAEC past questions on all subjects for reference and review. It also sets questions on specified topics for teachers to assess the understanding of learners in those topics.

This application can combine required number of questions from different topics for a standard examination. You can print all questions and answers instantly and concurrently.
